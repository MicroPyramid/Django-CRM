==========
Django CRM
==========

Django CRM is an opensource CRM on Django framework developed and maintained by `Micropyramid`_. It has all the basic features of Sales and Marketing to begin with and customise to meet industry needs.

.. _Micropyramid: https://micropyramid.com/ 

We welcome code contributions, suggestions, and feature requests via github. Source Code is available in `Micropyramid Repository`_.

.. _Micropyramid Repository: https://github.com/MicroPyramid/Django-CRM.git
